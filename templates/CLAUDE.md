# CLAUDE.md

This project follows the operating contract in `AGENTS.md`. Treat `AGENTS.md` as the single source of truth for how to work here.

Before starting any task:

1. Read `AGENTS.md` for the operating contract.
2. Read `memory-bank/activeContext.md` for current state (then `projectbrief.md`, `systemPatterns.md` only as needed).

After meaningful work, update `memory-bank/activeContext.md` and `progress.md` — this is the Compound step. Leave the system smarter than you found it.

Git: stage or edit only from a sandboxed agent; hand the human the commit command (see Git Hygiene in `AGENTS.md`).
